# AbiHomeTest

### Setup directly
- Install MongodDB, make sure you have Mongo instance is running.

- Install Nodejs

- Clone the repo

- From terminal, cd `AbiHome-test`

- Run `npm install` to install all dependencies 
- Run `npm start` to start the application

#### Running Unit test

- Run `npm test`
